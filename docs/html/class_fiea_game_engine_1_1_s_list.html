Detailed Description

template<typename T>
class FieaGameEngine::SList< T >

A class which defines a templated LinkedList Each "node" is made up of templated data and a next pointer which is a reference to the next list's element front refers to the node at the front of the list back refers to the node at the back of the list size refers to the size of the list
