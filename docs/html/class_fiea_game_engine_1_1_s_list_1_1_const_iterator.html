Detailed Description

template<typename T>
class FieaGameEngine::SList< T >::ConstIterator

A templated class which defines a non-inheritable ConstIterator Stores a reference the owning list and the node this iterator is referring to The difference between this and a regular iterator, is that its node is const and cannot be mutated This will be mainly used to traverse the list
