Detailed Description

template<typename T>
class FieaGameEngine::Vector< T >::ConstIterator

A templated class which defines a non-inheritable Iterator Stores a reference to the owning vector and the index this iterator is referring to The difference between this and a regular iterator, is that its vector is const and cannot be mutated This will be mainly used to traverse the vector
